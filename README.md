Aplicación para almacenar información de objetos multimedia  (películas, discos, MP3), el formato puede ser  (wav, mp3, midi, avi, mov, mpg, cdAudio, dvd).
La clase Película HEREDA de la clase multimedia y contiene 2 atributos mas (ator principal y actriz principal) y sobre escribe el método toString de la clase Multimedia.
La clase listaMultimedia contiene un atributo de tipo array  de objetos multimedia y  un entero , el array almacena objetos de tipo multimedia.
